# Установка Renga Sverchok Nodes

## Быстрая установка

### Шаг 1: Копирование файлов

Скопируйте папку `renga` из `sverchok_nodes/renga` в папку Sverchok:

```
C:\Users\ВАШЕ_ИМЯ\AppData\Roaming\Blender Foundation\Blender\5.0\scripts\addons\sverchok-master\nodes\renga
```

**Важно:** Скопируйте именно папку `renga` со всеми файлами внутри!

### Шаг 2: Перезапуск Blender

1. Полностью закройте Blender
2. Запустите Blender снова
3. Откройте консоль (Window > Toggle System Console) и проверьте сообщения:
   - Должно быть: `Renga nodes: Найдена папка sverchok-master: ...`
   - Должно быть: `Renga nodes: ✓ Секция Renga добавлена в index.yaml`

### Шаг 3: Перезагрузка аддона Sverchok

1. Edit > Preferences > Add-ons
2. Найдите "Sverchok" в списке
3. Снимите галочку (отключите аддон)
4. Поставьте галочку обратно (включите аддон)

### Шаг 4: Проверка

1. Откройте Sverchok (Shift+A в редакторе нод)
2. Найдите категорию **"Renga"** в меню добавления нод
3. Должны быть доступны три ноды:
   - **Renga Connect** - подключение к TCP серверу Renga
   - **Renga Create Columns** - создание колонн в Renga
   - **Renga Get Walls** - получение стен из Renga

## Автоматическая установка

Все происходит автоматически при копировании папки `renga` и перезапуске Blender:

1. ✅ Ноды автоматически регистрируются
2. ✅ Секция Renga автоматически добавляется в меню Sverchok
3. ✅ Никаких дополнительных действий не требуется

## Требования

- **Blender 5.0** (или 4.2+)
- **Sverchok** аддон установлен и включен
- **Renga** с установленным плагином RengaGH (TCP сервер)

## Устранение проблем

### Ноды не появляются в меню

1. Проверьте консоль Blender на наличие ошибок
2. Убедитесь, что папка `renga` скопирована в правильное место
3. Перезагрузите аддон Sverchok (Edit > Preferences > Add-ons)
4. Перезапустите Blender полностью

### Ошибки при регистрации

Если видите ошибки типа "already registered", это нормально - ноды уже зарегистрированы. Просто перезагрузите аддон Sverchok.

## Структура файлов

```
sverchok_nodes/renga/
├── __init__.py              # Автоматическая регистрация и добавление в меню
├── renga_connect.py         # Нода подключения к Renga
├── renga_create_columns.py  # Нода создания колонн
├── renga_get_walls.py       # Нода получения стен
├── renga_client.py          # TCP клиент
├── connection_protocol.py   # Протокол связи
└── commands.py              # Команды для Renga
```

Все файлы должны быть скопированы вместе!
